# Privacy Policy — Zapret Helper

## English

Zapret Helper browser extension does not collect, store, or transmit any user data.

**Data handling:**
- The extension communicates only with the local Zapret GUI server (127.0.0.1:8080)
- No data is sent to external servers
- No analytics, telemetry, or tracking is performed
- No cookies or local storage are used for tracking purposes

**Permissions:**
- `activeTab`: Used to detect the current website URL
- `storage`: Used to store the Zapret GUI address setting locally

**Third parties:**
- The extension does not share data with any third parties
- No advertising or tracking scripts are included

**Changes:**
Any changes to this privacy policy will be updated in the extension's source code on GitHub.

## Русский

Расширение Zapret Helper не собирает, не хранит и не передаёт никакие данные пользователей.

**Обработка данных:**
- Расширение общается только с локальным сервером Zapret GUI (127.0.0.1:8080)
- Никакие данные не отправляются на внешние серверы
- Аналитика, телеметрия и отслеживание не ведутся
- Cookies и localStorage не используются для отслеживания

**Разрешения:**
- `activeTab`: Используется для определения URL текущего сайта
- `storage`: Используется для хранения настройки адреса Zapret GUI

**Третьи стороны:**
- Расширение не передаёт данные третьим сторонам
- Рекламные и отслеживающие скрипты отсутствуют

**Изменения:**
Любые изменения в политике конфиденциальности публикуются в исходном коде расширения на GitHub.
