# Zapret Helper — Listing

## Chrome Web Store

### Название
Zapret Helper

### Краткое описание (макс 132 символа)
Быстрое добавление заблокированных сайтов в списки обхода DPI через Zapret GUI

### Описание
Zapret Helper — расширение для быстрого добавления заблокированных сайтов в списки обхода DPI.

Расширение работает в паре с Zapret GUI — графическим интерфейсом для управления zapret, инструментом обхода DPI-блокировок на Windows.

**Как использовать:**
1. Установите и запустите Zapret GUI
2. Установите расширение Zapret Helper
3. Откройте заблокированный сайт
4. Нажмите иконку расширения на панели инструментов
5. Нажмите «Добавить в обход» — домен автоматически добавится в список
6. Перезагрузите страницу — сайт должен открыться

**Возможности:**
• Добавление текущего сайта в список обхода одним кликом
• Удаление сайтов из списка
• Выбор списка (общие, пользовательские, исключения)
• Просмотр добавленных доменов
• Настройка адреса Zapret GUI

**Требования:**
• Zapret GUI должен быть запущен на компьютере
• Доступ к localhost:8080

**Прозрачность:**
Расширение не собирает никаких данных. Все операции выполняются локально на вашем компьютере через Zapret GUI. Расширение не имеет доступа к интернету, не отправляет данные на внешние серверы.
